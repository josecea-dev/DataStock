# DataStock

Sistema de inventario desarrollado con:

- ASP.NET Web API
- SQL Server
- JavaScript (Fetch API)
- HTML y CSS

## Funcionalidades

- Gestión de usuarios
- Login con hash de contraseñas
- Control de roles (Administrador / Empleado)
- Gestión de productos
- Registro de movimientos (entrada / salida)
- Búsqueda dinámica

## Tecnologías

- C#
- SQL Server
- JavaScript
- HTML/CSS

## Cómo ejecutar

1. Clonar repositorio
2. Ejecutar script SQL para crear la base de datos
3. Configurar cadena de conexión en appsettings.json
4. Ejecutar la API
5. Abrir el frontend
